# Amazon-Sales-Prediction-Analysis-Power-Bi-
This project focuses on analyzing and predicting Amazon sales performance using interactive dashboards built in Microsoft Power BI. The objective is to transform raw sales data into meaningful insights and forecast future sales trends to support data-driven decision-making.

**🚀 Project Overview**

The dataset includes key sales attributes such as order date, product category, region, sales amount, and customer details. Using Power BI, the data is cleaned, transformed, and visualized to identify patterns, trends, and performance metrics.

**🎯 Objectives**
Analyze historical sales data
Identify top-performing products and regions
Track monthly and yearly sales trends
Predict future sales using trend analysis
Build an interactive and user-friendly dashboard

**🛠️ Tools & Technologies**
Microsoft Power BI (Data Visualization & Dashboarding)
Excel / CSV (Dataset)
DAX (Data Analysis Expressions)
Power Query (Data Cleaning & Transformation)

**📈 Key Features**
Sales Overview Dashboard (KPIs: Total Sales, Profit, Orders)
Region-wise and Category-wise Analysis
Time Series Analysis (Monthly/Yearly Trends)
Top Products & Customer Insights
Sales Forecasting using built-in Power BI analytics

**🔍 Insights Generated**
Identified peak sales periods and seasonal trends
Highlighted high-revenue product categories
Compared regional performance for better strategy planning
Forecasted future sales trends for business growth

**📌 Conclusion**

This project demonstrates the power of business intelligence tools like Microsoft Power BI in transforming raw data into actionable insights. It enhances analytical thinking and provides practical exposure to real-world data analysis and visualization.
